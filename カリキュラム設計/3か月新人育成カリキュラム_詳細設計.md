# 3か月新人育成カリキュラム 詳細設計

## 位置づけ

この資料は、学習コンテンツ一覧を実施運用できる粒度まで詳細化したものです。  
各テーマについて、具体的な学習内容、概算時間、理解確認の小テスト、講師共通の評価基準を整理しています。

## 運用方針

| 観点 | 方針 |
| --- | --- |
| 学習の進め方 | 学習 → 演習 → 小テスト → フィードバック の順で進める |
| 理解確認 | 大きな区切りごとに小テストを入れ、中間確認を行う |
| 評価の揺れ防止 | 講師ごとの感覚ではなく、共通の観点で判定する |
| 合否の考え方 | 1回で完了を求めず、再提出・再テスト前提で定着を優先する |
| AI活用 | 調査・実装・レビューで使わせるが、理解確認は本人説明を重視する |

## 評価レベル定義

| 評価 | 定義 |
| --- | --- |
| A | 自力で説明・実施でき、応用質問にも対応できる |
| B | 基本は理解しており、軽い補助で実施できる |
| C | 用語は知っているが、実施や説明に詰まりがある |
| D | 理解不足で、再学習が必要 |

## 共通の判定観点

| 観点 | 見るポイント |
| --- | --- |
| 理解 | 用語を暗記ではなく、自分の言葉で説明できるか |
| 再現 | 手順や実装を再現できるか |
| 説明 | なぜその方法を取るかを説明できるか |
| 品質 | 漏れ、誤り、雑さが少ないか |
| 自走 | 詰まった時に調査・切り分け・相談ができるか |

## 講師共通の評価シート案

| 項目 | A | B | C | D |
| --- | --- | --- | --- | --- |
| 理解説明 | 自分の言葉で構造的に説明できる | 基本説明はできる | 用語説明が曖昧 | 説明できない |
| 実装再現 | 自力で再現できる | 軽い補助で再現できる | 多くの補助が必要 | 再現できない |
| 品質意識 | テスト、確認、記録まで自然にできる | 指示があれば対応できる | 抜け漏れが多い | 品質観点が弱い |
| 自走力 | 調査、切り分け、相談が適切 | 一部支援が必要 | 詰まると停止しやすい | 自走が難しい |
| 対人対応 | 報告・説明・受け答えが安定 | 基本対応はできる | 不足が目立つ | 大きな課題がある |

## 講師介入の基本方針

| 状況 | 講師の介入方針 |
| --- | --- |
| 初回導入時 | ゴール、資料の見方、提出物、質問方法を最初に説明する |
| 小テスト前 | 論点整理と不明点確認だけ行い、答えは先に渡しすぎない |
| 小テスト後 | 結果よりも「どこで詰まったか」を言語化させる |
| 遅れ発生時 | 感覚で叱責せず、原因を「理解不足」「作業量」「進め方」に分けて見る |
| 再学習時 | 全部やり直しではなく、弱い論点だけを指定して戻す |

## 遅れが発生している場合の進め方

| 遅れのタイプ | 見分け方 | 進め方 |
| --- | --- | --- |
| 理解不足 | 説明できない、同じ箇所で毎回止まる | 講師が15-30分で要点整理し、再度ミニ課題を実施する |
| 作業過多 | 理解はあるが提出が終わらない | 課題を分割し、必須範囲だけ先に完了させる |
| 進め方の問題 | 手を付ける順が悪い、調査に時間を使いすぎる | 作業順、調べ方、相談タイミングを具体化する |
| 品質不足 | 動くが雑、説明できない、再現性が低い | 再提出を求め、チェック観点を明示する |
| メンタル停滞 | 質問が減る、進捗報告が曖昧になる | 1on1で詰まりと不安を切り分け、短い成功体験を作る |

| 中 | 提案型コミュニケーション、案件拡張視点、インフラ発展項目 | 発展的なケーススタディ |

## 学習不足と判断する基準

| 判定観点 | 学習不足のサイン |
| --- | --- |
| 理解 | 用語は知っているが、自分の言葉で説明できない |
| 再現 | 同じ課題を再度実施すると再現できない |
| 応用 | 少し条件が変わると手が止まる |
| 説明 | なぜその実装・判断なのかを言えない |
| 品質 | テスト、確認、ドキュメントが毎回抜ける |
| 自走 | エラー時に調査せず停止する、または丸投げする |

## 再学習の進め方

| ステップ | 内容 |
| --- | --- |
| 1 | どの論点が弱いかを講師が明示する |
| 2 | 元資料の該当範囲だけ再学習させる |
| 3 | 同難度か少し簡単なミニ課題で再確認する |
| 4 | 口頭説明をさせて、理解と暗記を分けて確認する |
| 5 | B以上になったら元の流れへ復帰させる |

## 再学習に戻すか先へ進めるかの判断

| 状況 | 判断 |
| --- | --- |
| 小テストがCでも、補助ありで説明と再現が可能 | 先へ進めつつ並行補強 |
| Dで、次の学習に必要な前提が欠けている | 再学習へ戻す |
| 実装はできるが説明できない | 口頭確認を追加して補強 |
| 説明はできるが実装できない | ハンズオン課題を追加して補強 |
| 複数回同じ論点で止まる | 講師主導で短期補講を入れる |
