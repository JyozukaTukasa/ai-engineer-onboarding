# 3か月新人育成カリキュラム 月別時間割案

## 前提

- 期間区分: 1ヶ月目、2ヶ月目、3ヶ月目
- 目的: 週単位より上位の粒度で、各月に何を重点配置するかを時間割として見える化する
- 想定: 1日 9:00-18:00、昼休憩 12:00-13:00、短休憩 14:00-14:15

## 3ヶ月の流れ

```mermaid
flowchart LR
    A["1ヶ月目<br/>下流工程の土台"] --> B["2ヶ月目<br/>下流実践から上流接続"]
    B --> C["3ヶ月目<br/>上流実践と顧客対応"]
    A -.並行.-> D["AI活用<br/>調査・実装・レビュー"]
    B -.並行.-> D
    C -.並行.-> D
```
